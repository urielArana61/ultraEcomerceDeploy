# 🛍️ ultaEcomerce

Proyecto de tienda en línea desarrollado con React + Vite. Utiliza tecnologías modernas para construir una experiencia rápida, escalable y estética.

---

## 🚀 Stack Tecnológico

- ⚛️ **React** v19.1  
- 🌐 **React Router DOM** v7.7.1  
- 🔄 **TanStack Query**  
- 🎨 **Tailwind CSS** v4.1.11  
- 🧩 **shadcn/ui**  
- 🌐 **Axios**
